# Gett Application
 
### In the root directory:

##### Run `yarn install`

Install app dependencies.<br>

##### Run `yarn start`

Runs the app.<br>

### In the client directory:

#### Change to local IP
Go to *src/containers/Chat.js* at line 7 and change the IP in the URL to the server's local IP.

##### Run `yarn install`

Install app dependencies.<br>

##### Run `yarn start`

Runs the app.<br>
