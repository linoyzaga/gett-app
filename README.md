# Gett Application
 
### In the root directory, you need to run:

##### `yarn install`

Install app dependencies.<br>

##### `yarn start`

Runs the app.<br>

### In the client directory, you need to run:

#### Change to local IP
Go to *Chat.js* at line 7 and change the IP in the URL to **your** local IP.

##### `yarn install`

Install app dependencies.<br>

##### `yarn start`

Runs the app.<br>
